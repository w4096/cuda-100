This folder contains some CUDA kernels I have implemented to solve the problems on LeetGPU.com.

## Run the Solutions on Your Local Machine

Set `PYTHONPATH` to the root directory of this repository

```bash
cd path/to/repo
export PYTHONPATH=$(pwd)
```
Install the required dependencies:

```
pip install torch
```

Run the kernel with:

```bash
python ./run.py
```
